# Dicas Markdown

As "#" são títulos no repositório.
Duas hash (##) significa um subtítulo. Você pode incluir mais subtítulos. 

# Título 

## Substitulo 

### Subtitulo

# Destaque

\*\*negrito\*\* - **negrito**
\_italico\_ - _italico_
\~\~riscado\~\~ - riscado 

# Listas

1. primeiro
2. segundo
3. terceiro

ou

* item 1
* item 2
* item 3
